const promise = Promise.resolve();
let i = 0;
promise.then(() => {
  i += 1;
  console.log(i);
});
promise.then(() => {
  i += 1;
  console.log(i);
});
我们来看一下这段代码，就是说promise.then()这是两个异步操作的回调函数，在Java语言多线程中，会出现抢锁，即可能出现两次打印都是2，比如下面的抢到锁i+1得2时候，还没赋值就是上面的代码块抢锁赋值得到2输出了，再释放锁下面代码块赋值为2，即打印了两个2
在js中，上面代码块先进入了回调函数队列，后面的代码块后进入回调队列，所以输出结果是可预测的，即输出2、3


插一嘴题外话，.then函数是微任务，而setTimeOut、setInterval、script这些是宏任务。
默认是先执行一个宏任务（script），再排序执行宏任务中的微任务。然后执行下一个宏任务任务。
console.log(1)
setTimeout(()=>console.log(2),0)
Promise.resolve().then(()=>console.log(3))
console.log(4)

打印顺序为1432，先打印script中的1、4因为没有回调(先执行同步代码块)，然后执行微任务，即.then，最后是宏任务setTimeOut
